# Plan: 06-02 Web Dashboard & Clinical Viewer

## Requirements Covered
- **DASH-01**: List cases and synced triage results.
- **DASH-02**: Clinical visualizer for Grad-CAM and Report download.

## Goals
Implement a **Specialist Dashboard** for remote review of referred oral cancer cases.

## Dependencies (Wave 0)
- `SyncService` (Phase 5 complete).
- Dashboard static build config in FastAPI.

## Tasks (Wave 1)

1. **FastAPI STATIC Assets (Backend)**
    - Implement `Mount` in `main.py` for `/dashboard` serving.
    - Path: `dist/` (react build artifacts).
2. **Dashboard UI (React/Vite/Tailwind)**
    - Build a modern **Case Dashboard**.
    - Features: Triage Status (Normal/Malignant), Confidence Gauges, Uncertainty scores.
    - Gallery Viewer: Raw image vs Enhanced side-by-side with Grad-CAM toggle.
    - One-click PDF Report download.
3. **CORS Configuration (FastAPI Middleware)**
    - Add `CORSMiddleware` to `main.py`.
    - Allow development from `localhost:5173`.

## Verification Instructions
